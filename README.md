# Fikus — Система автоматизации продаж

Полная инфраструктура: лендинг → заявки → CRM-бот → автоматический поиск клиентов → Instagram-рассылка.

## 1. Сайт — `../` (Next.js)

**Где:** `/fikus-website/` (корень репозитория)
**Хостинг:** GitHub Pages
**Деплой:** автоматически при push в `main` через `.github/workflows/deploy.yml`

| Файл / папка | Что делает |
|---|---|
| `app/[locale]/page.tsx` | Главная страница с мультиязычностью |
| `components/sections/` | Секции: Hero, Portfolio, Services, FAQ, Contact, и т.д. |
| `messages/*.json` | Переводы: ru, en, ka, uz |
| `i18n/routing.ts` | Настройка маршрутов по локали |
| `public/` | Картинки кейсов, логотипы |

Контактная форма отправляет `POST /webhook` на Alert Bot с полями: `name`, `email`, `company`, `message`, `budget`.

## 2. Alert Bot — `fikus-alert-bot/`

**Хостинг:** Render (free plan)
**Стек:** FastAPI + python-telegram-bot + Google Sheets API

### Как запускается

```
uvicorn main:app --host 0.0.0.0 --port $PORT
```

При старте: инициализирует Sheets, запускает Telegram-бот polling, шлёт дайджест последних заявок.

### Файлы

| Файл | Роль |
|---|---|
| `main.py` | FastAPI сервер. Принимает `/webhook` с сайта, создаёт лид в Sheets, уведомляет бот |
| `bot.py` | Telegram-бот: карточки заявок с кнопками, управление статусами, партнёры, заметки |
| `sheets.py` | Все операции с Google Sheets (чтение/запись лидов, партнёров, хантер-лидов) |
| `config.py` | Переменные окружения, названия вкладок Sheets |
| `hunter_utils.py` | Форматирование карточек хантер-лидов и клавиатур (общее для bot.py и хантеров) |

### Кнопки в карточке заявки (входящей)

- **На тест** → статус «На тесте»
- **Отложить** → статус «Отложено»
- **Отклонить** → статус «Отклонено»
- **→ Партнёры** → диалог, переводит в вкладку «Партнёры»
- **Заметка** → добавляет заметку с меткой времени
- **Таблица** → открывает Google Sheets

### Кнопки в карточке хантер-лида (холодного)

- **📞 Связались** → статус «Связались»
- **✅ В pipeline** → статус «В pipeline»
- **📸 Написать в Instagram** → автоматически шлёт DM через `instagram_dm.py`
- **⏳ Пропустить 30д** → статус «Отложено»
- **❌ Не целевой** → статус «Не целевой»
- **📍 Открыть на Maps** → ссылка на Google Maps

### Команды бота

| Команда | Что делает |
|---|---|
| `/pending` | Все заявки со статусом «Новая» и «Отложено» |
| `/partners` | Список активных партнёров |
| `/help` | Инструкция |
| `/cancel` | Выйти из текущего диалога |

### Переменные окружения

```env
BOT_TOKEN=           # токен Telegram-бота
CHAT_ID=             # ID чата/группы куда бот шлёт уведомления
TOPIC_ID=            # ID топика в группе (если есть)
SECRET_KEY=          # секрет для защиты /webhook
SPREADSHEET_ID=      # ID Google Sheets таблицы
GOOGLE_CREDENTIALS_JSON=  # JSON сервис-аккаунта (для Render)
HUNTER_CHAT_ID=      # чат для хантер-уведомлений (по умолчанию = CHAT_ID)
HUNTER_TOPIC_ID=     # топик для хантер-уведомлений

# Instagram DM (необязательно)
IG_USERNAME=         # логин Instagram аккаунта для рассылки
IG_PASSWORD=         # пароль
IG_DAILY_LIMIT=20    # максимум DM в день
```

## 3. Hunter система

Автоматический поиск бизнесов без сайта по Google Maps. Работает через GitHub Actions — парсит места, фильтрует, сохраняет в Sheets, шлёт уведомления в Telegram.

### Scraper (`scraper/`)

| Файл | Роль |
|---|---|
| `maps.py` | Playwright-скрапер Google Maps. Города: Georgia (12 городов), Uzbekistan (9 городов). Собирает: название, телефон, сайт, Instagram, рейтинг, отзывы |
| `scorer.py` | Фильтр лидов: рейтинг ≥ 4.3, отзывов ≥ 50, нет нормального сайта (Wix/Tilda не считается) |
| `niches.py` | 23 ниши: кафе, рестораны, отели, красота, фитнес, стомат, туризм и т.д. |

### Хантер-скрипты

#### `hunter.py` — Грузия, еда и напитки
Запускает 5 запросов (кафе, рестораны, кофейни, винные бары, пекарни) по 12 городам Грузии. Сохраняет в вкладку `Hunter_Leads`.

#### `hunter_biz.py` — Грузия, бизнес-ниши
Запускает 12 бизнес-запросов (отели, красота, фитнес, авто, туризм и т.д.) по Грузии. Сохраняет в `Hunter_Biz`.

#### `hunter_niche.py` — универсальный (одна ниша × страна)
Управляется через env:
```bash
NICHE_KEY=hotels COUNTRY=Uzbekistan MAX_RESULTS=200 python hunter_niche.py
```
Сохраняет в единую вкладку `«Лиды»`.

#### `hunter_uz.py` — Узбекистан, все ниши
Автоматически проходит по 14 нишам для 9 городов Узбекистана. Отправляет Instagram DM. Сохраняет в `«Лиды»` с пометкой страны.

```bash
# Запустить все дефолтные ниши
python hunter_uz.py

# Только выбранные ниши
NICHES_UZ=cafes,hotels,beauty python hunter_uz.py

# Настроить объём
MAX_RESULTS_PER_NICHE=100 python hunter_uz.py
```

### Алгоритм скрапинга

```
Для каждого города × каждого запроса:
  1. Открыть Google Maps с координатами города
  2. Собрать 25 ссылок из списка (фаза 1)
  3. Зайти на каждую страницу места → спарсить детали (фаза 2)
  4. Передать в scorer → отфильтровать слабых
  5. Сохранить новых в Google Sheets
  6. Если есть Instagram → отправить DM
```

### Критерии фильтрации (scorer.py)

| Условие | Действие |
|---|---|
| Рейтинг < 4.3 | Отсеять |
| Отзывов < 50 | Отсеять |
| Есть нормальный сайт | Отсеять — нам не нужны |
| Сайт на Wix/Tilda/Jimdo/конструкторе | Пропустить → лид «Сайт на конструкторе» |
| Только Instagram, сайта нет | Лид → «Только Instagram» |
| Нет ни сайта, ни Instagram | Лид → «Топовое заведение без сайта» |

### Оценка бюджета

| Отзывов | Оценка |
|---|---|
| 500+ | $2,000 – $4,000 |
| 200–499 | $1,500 – $3,000 |
| 100–199 | $1,000 – $2,000 |
| 50–99 | $800 – $1,500 |

## 4. Instagram DM — `instagram_dm.py`

Автоматически отправляет персонализированные DM лидам у которых есть Instagram.

- Использует `instagrapi` (неофициальный API)
- Дневной лимит: `IG_DAILY_LIMIT` (по умолчанию 20 DM)
- Сессия кешируется в `ig_session.json` — не нужно логиниться каждый раз
- Пауза между DM: 1–2 минуты
- После отправки — статус лида меняется на «Написали в Instagram»

**Важно:** использовать прогретый отдельный аккаунт, без 2FA.

## 5. GitHub Actions — `.github/workflows/`

| Workflow | Триггер | Что делает |
|---|---|---|
| `deploy.yml` | Push в `main` | Билдит Next.js и деплоит на GitHub Pages |
| `hunter.yml` | Вручную (+ можно по расписанию) | Запускает `hunter.py` для Грузии (еда) |
| `hunter_biz.yml` | Вручную | Запускает `hunter_biz.py` для Грузии (бизнес) |
| `hunter_niche.yml` | Вручную, выбор ниши и страны | Одна ниша × одна страна |
| `hunter_all.yml` | Вручную | Все 23 ниши параллельно (matrix strategy) |

Расписание пока закомментировано во всех хантерах — включить когда будет проверено качество данных.

### Секреты GitHub (Settings → Secrets)

```
BOT_TOKEN
CHAT_ID
TOPIC_ID
HUNTER_CHAT_ID
HUNTER_TOPIC_ID
SPREADSHEET_ID
GOOGLE_CREDENTIALS_JSON
```

## 6. Google Sheets — структура

| Вкладка | Что хранит | Кто пишет |
|---|---|---|
| `Заявки` | Входящие заявки с сайта | `main.py` /webhook |
| `Партнёры` | Партнёрская база | bot.py → диалог «→ Партнёры» |
| `Лиды` | Все холодные лиды от хантеров | `hunter_niche.py`, `hunter_uz.py` |
| `Hunter_Leads` | Legacy — еда Грузия | `hunter.py` |
| `Hunter_Biz` | Legacy — бизнес Грузия | `hunter_biz.py` |

Колонки в `«Лиды»`: ID · Дата · Ниша · Страна · Город · Название · Maps · Сайт · Instagram · Телефон · Рейтинг · Отзывов · Почему лид · Бюджет · Статус · Дата контакта · Заметки

## Быстрый старт

### Alert Bot (локально)
```bash
cd fikus-alert-bot
pip install -r requirements.txt
cp .env.example .env   # заполнить переменные
uvicorn main:app --reload
```

### Хантер (локально)
```bash
pip install -r requirements-hunter.txt
playwright install chromium

# Грузия — еда
python hunter.py

# Узбекистан — все ниши
python hunter_uz.py

# Одна ниша, любая страна
NICHE_KEY=hotels COUNTRY=Uzbekistan python hunter_niche.py
```

### Сайт
```bash
cd ..   # в корень fikus-website
npm install
npm run dev   # http://localhost:3000
```
